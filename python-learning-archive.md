# Python Learning Project

This project is designed to help JavaScript developers learn Python syntax and concepts.

## Features

- **syntax_comparison.py**: Side-by-side comparison of Python and JavaScript syntax
- **exercises.py**: Practice exercises for Python learning

## Requirements

See `requirements.txt` for dependencies.

## Installation

```bash
pip install -r requirements.txt
```

## Running

### Syntax Comparison
```bash
python syntax_comparison.py
```

### Exercises
```bash
python exercises.py
```

## Learning Resources

This project focuses on practical Python examples for those already familiar with JavaScript.

For face detection with camera, see the `face-detection/` project in the parent directory.
